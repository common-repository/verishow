== Description ==
VeriShow is the world's first multimedia platform for delivering live help and real-time sales support to website visitors. The WordPress plugin enables you to interact with site visitors in real time using live chat, voice and video and access a host of collaborative tools, such as document, image and product-video sharing, a second-camera input and many others. You and your visitors can annotate those in real-time, for example to point out a feature on a product document or image. 

For more information visit: http://www.verishow.com/?from=wordpress

= Features =
* Chat, video & audio conferencing
* Document sharing, joint form-filling
* Whiteboards, co-editing documents
* Simple, 5 minute implementation
* Fully hosted

= Benefits =
* Communicate with your site & blog visitors 
* Convert up to 3 times more visitors
* Provide effective customer service & sales
* Promote products with multimedia tools
* Provide in-store experience online.

== Installation ==
1.	Download the VeriShow plugin zip file, extract it and upload the extracted folder to your /wp-content/plugins/ folder
2.	Log into your Wordpress admin, click Plugins, find the VeriShow plugin and click Activate. 
3.	At bottom left, click Settings > VeriShow and follow the instructions there.

== Frequently Asked Questions ==

= Do I need a VeriShow account? =
Yes. Please visit http://www.verishow.com/?from=wordpress and open an account - Free and premium options available.

= How do I prevent the button from appearing on a particular page? =
If you have installed the button on your site but want to exclude a particular page from displaying the button, please follow these steps:

= Steps: =
* Go to Wordpress admin "pages" and select the particular page.
* Click on the "HTML" tab.
* Paste the following code at the end of the page and save the page.
<code>
<code>
<script type="text/javascript">
var elem=document.getElementById("hbrlink");
elem.style.display = 'none'; 
elem.style.visibility = 'hidden';
</script> 
</code>
</code>
